## 1. USCIS Requirements — 8 CFR 103.2(b)(3)

**Source:** law.cornell.edu/cfr/text/8/103.2 + USCIS Policy Manual (verified 2025)

### What's required
- **Full translation word-for-word** — no summaries, no "key fields only"
- **Everything visible must be translated**: stamps, seals, marginal notes, even faint handwriting
- **Translator's certification block** — must include:
  - Full name of translator
  - Signature (handwritten for paper filings)
  - Mailing address
  - Date
  - Statement of competency and accuracy

### Sample certification statement (USCIS-accepted language)
```
I, [name], hereby certify that I am competent to translate from [language] into English
and that the foregoing is a complete and accurate translation of the attached document.
```

### What is NOT required
- Notarization (optional unless another authority specifically requests it)
- Government license or ATA membership
- Blanket certificate for multiple documents — **each document needs its own cert**

### Applies to all forms
I-130, I-485, I-539, I-765, N-400, I-131, I-751, I-829, VAWA, asylum, waivers, and all supporting documents.

## 2. Document Type Standards

### 2a. Ukrainian Passport (biometric, issued 2015+)
Fields to translate:
- Surname / Given name / Patronymic (відчество — translate as-is, note "patronymic")
- Date of birth / Place of birth
- Sex / Nationality / Citizenship
- Document number (alphanumeric — keep unchanged, do not transliterate)
- Issue date / Expiry date
- Issuing authority
- MRZ lines — translate as-is (do not modify)

**KMU 2010 transliteration standard** applies for names (Cabinet Resolution #55).

### 2b. Ukrainian Birth Certificate (modern, 2000+)
Fields:
- Child's full name (surname, given name, patronymic)
- Date of birth / Place of birth (city, region, Ukraine)
- Sex
- Father's full name / Mother's full name
- Act (registry) number
- Date of registration
- Issuing authority (Civil Registry Office / DRACS)
- Seal text (translate: "Civil Registry Office of [city], [region], Ukraine")

### 2c. Soviet Birth Certificate (УРСР, issued 1920–1991)
**Critical extra fields vs modern:**
- **Ethnic nationality** (Національність / Национальность): translate literally —
  "Ukrainian", "Jewish", "Russian", "Armenian" etc. — DO NOT omit
- **Republic name**: "Ukrainian Soviet Socialist Republic" — add note: "(now Ukraine)"
- **Registry book number** (Книга реєстрацій №): translate as "Registry Book No."
- **Place names**: many cities renamed — translate original + add note of modern name
  - Example: "Сталіно" → "Stalino (now Donetsk, Ukraine)"
  - Example: "Ворошиловград" → "Voroshilovgrad (now Luhansk, Ukraine)"
- **Patronymic**: translate as-is, USCIS does not require it but it must be present

**Special note for immigration timeline:**
Person born in USSR (1920–1991) → later lived in independent Ukraine/Russia →
now in USA. Birth certificate is from УРСР. USCIS accepts Soviet-era documents.
Translator must note document era clearly in the translation header.

### 2d. Soviet Birth Certificate (РСФСР, issued 1920–1991)
Same as УРСР but:
- Republic name: "Russian Soviet Federative Socialist Republic" (add: "(now Russia)")
- Transliteration: GOST 7.79-2000 System B for Russian names

### 2e. Marriage Certificate (Свідоцтво про шлюб)
Fields:
- Husband's full name (surname + given + patronymic before and after marriage)
- Wife's full name (same)
- Date of marriage
- Place of marriage (city, region, country)
- Act number / Registry date
- Issuing authority (DRACS / ZAGS)
- Seal text

### 2f. Driver's License (Ukrainian, credit-card format)
Fields:
- Surname / Given name / Patronymic (fields 1, 2, 3 on document)
- Date of birth (4a) / Place of birth (4b)
- Issue date (4c) / Expiry (4b on some versions)
- Document number (5)
- Issuing authority (4d)
- Vehicle categories (9 — e.g., B, C1)
- Restrictions if any

### 2g. Diploma / Academic Certificate (Ukrainian university)
Fields:
- Graduate's full name
- Degree title: "Bachelor of [field]" / "Master of [field]" / "Specialist in [field]"
- Specialty code and name
- Institution full official name (in English)
- Date of graduation / Date of issue
- Document number (series + number)
- Rector's name + signature line
- Dean's name + signature line
- University seal text (translate fully)

### 2h. Police Clearance Certificate (довідка про несудимість)
Fields:
- Full name / Date of birth / Place of birth
- Record type: "No criminal record" or describe record
- Document number / Issue date / Valid until
- Issuing authority (Ministry of Internal Affairs / police department)

## 7. Notes on Patronymic Names

Ukrainian/Russian names have 3 parts: Surname + Given name + Patronymic (батьківське ім'я / отчество).
- US systems often don't have a "patronymic" field
- USCIS: include in translation, label as "Patronymic (middle name)"
- If passport only has 2 fields (Last, First) — translate what's there, note "patronymic not shown on this document"
- For USCIS forms asking for "Middle Name" — patronymic goes there
