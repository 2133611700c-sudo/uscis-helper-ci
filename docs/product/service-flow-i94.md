# Service Flow — I-94 Helper

Updated: 2026-04-30

## Flow

1. User clicks `I-94`
2. User sees short explanation of what I-94 is used for
3. Button opens official CBP I-94 tool
4. User saves/downloads record
5. User uploads I-94 or enters I-94 number into related flows
6. System reuses I-94 data in SAMPLE/checklist builders

## Core Official Source

- https://i94.cbp.dhs.gov/
