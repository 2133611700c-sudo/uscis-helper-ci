# Service Flow — Re-Parole / U4U Package Helper

Updated: 2026-04-30

## Goal

Guide a user through document collection, field gathering, and sample/checklist preparation for re-parole related self-help.

## Flow

1. User clicks `Re-Parole`
2. User chooses scenario:
   - parole expires soon
   - already filed / have receipt
   - biometrics notice
   - payment not showing
   - received RFE
   - denied
   - need draft package help
3. User sees data/document checklist
4. User uploads passport, I-94, EAD, USCIS notice, and supporting files
5. System extracts document fields where possible
6. User answers missing manual questions
7. System generates:
   - completed SAMPLE
   - missing-fields checklist
   - source-linked filing packet guide
8. High-risk branch routes to attorney / DOJ accredited representative
