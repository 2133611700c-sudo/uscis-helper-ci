# Service Flow — EAD / Work Permit Helper

Updated: 2026-04-30

## Flow

1. User clicks `Work Permit`
2. User selects scenario:
   - initial
   - renewal
   - TPS-related
   - re-parole-related
3. User uploads passport, I-94, EAD, and USCIS notices if available
4. User answers manual questions
5. System produces:
   - completed SAMPLE
   - field map
   - missing-fields list
   - official source links
