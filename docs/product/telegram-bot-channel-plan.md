# Telegram Bot and Channel Plan

Updated: 2026-04-30

## Channel

Use the public channel for:
- verified updates
- FAQ posts
- official-source explainers
- scam warnings
- service announcements
- sponsored professional slots with disclosure

## Bot

Use the bot for:
- ask a question
- match existing answer
- return website link
- return official source
- collect new question
- start document/sample flow
- route high-risk topics

## High-Risk Topics

- denial
- RFE
- removal
- court
- asylum strategy
- criminal issue
- travel-risk issue
- fraud
- sponsor abuse

## Rule

No aggressive auto-replies in third-party groups.
Monitor first, admin review second, manual/light response only if allowed.
