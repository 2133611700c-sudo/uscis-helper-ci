# T3PS First Beta Users Script

## Кто может тестировать
- Только 3-5 доверенных тестеров.
- Без оплаты, без публичной рассылки.
- Это beta self-help инструмент подготовки черновика документов.

## Что нужно сделать тестеру
1. Открыть `/ru/services/tps-ukraine/start`.
2. Загрузить документ.
3. Проверить извлечённые поля и исправить ошибки.
4. Заполнить обязательные поля.
5. Нажать Generate и скачать ZIP.
6. Открыть PDF и проверить значения.
7. Нажать `Clear my data`.

## Что нельзя делать
- Не отправлять пакет без собственной финальной проверки.
- Не воспринимать сервис как юридическую консультацию.
- Не публиковать скриншоты с личными данными.
