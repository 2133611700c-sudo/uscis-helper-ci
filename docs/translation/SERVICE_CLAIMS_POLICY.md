# Service Claims Policy — Messenginfo v5.0

## FORBIDDEN phrases (never use in any public copy, UI, PDF, or email)
- "USCIS accepted"
- "guaranteed"
- "approved translation"
- "certified by AI"
- "instant certified translation"
- "100% accepted"
- "CERTIFIED COPY"
- "Certified Translation by Messenginfo"
- "translation agency"
- "law firm"

## ALLOWED phrasing
- "Prepared for USCIS-style submission"
- "AI-assisted draft for human review"
- "You review and sign if competent"
- "USCIS may request corrected or third-party translation"
- "Self-certified under 8 CFR §103.2(b)(3)"
- "Messenginfo is not a law firm"
- "The translator accepts full responsibility"

## Status terminology
| Status | Description |
|---|---|
| AI Draft | Machine-assisted draft. Not certified. Must be reviewed. |
| User-reviewed | User confirmed field values. Not automatically certified. |
| Human translator reviewed | A competent human checked/corrected the output. |
| Certified | Named human signed 8 CFR §103.2(b)(3) certification. |
| Notarized | Notary verifies identity/signature — not translation quality. |
