## Verification Methodology

### For each claim, verify against:

1. **USCIS Official Pages:**
   - uscis.gov/i-131 (Re-parole)
   - uscis.gov/i-765 (EAD)
   - uscis.gov/tps (TPS)
   - uscis.gov/ar-11 (Address reporting)

2. **USCIS Policy Manual:**
   - Search for relevant sections on parole, TPS, fraud detection

3. **Official Announcements:**
   - USCIS.gov news + press releases
   - Search for "U4U pause" OR "Ukraine parole" in 2024-2025 announcements

4. **Form Instructions:**
   - Read current I-131, I-765, I-821 instruction documents
   - Check for recent updates
