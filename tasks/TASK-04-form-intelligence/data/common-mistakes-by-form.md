# Common mistakes by form (from forensic audit research)

These are pre-validated mistakes from Facebook + Telegram community forensic audits. Use these verbatim in `common_mistakes_from_research` field.

DO NOT invent new mistakes — these are documented from real community pain points.

## I-131 (Re-parole / Travel Document)

- Checking the wrong box — Ukrainian re-parole filers should select Box 10.C, not "outside US" travel document boxes
- Filing too early — confusion about the 180-day window for re-parole submissions
- Form-only filing in 2026 — since August 2025, USCIS requires PERSONAL evidence (medical, family, employment), not just general "war in Ukraine" justification
- Incorrect parole expiration date entry — copying from wrong document
- Not including the original Travel Authorization document with the application
- Missing biometrics fee for first-time filers (the $85 fee is separate from the filing fee in some categories)

## I-765 (EAD / Work Permit)

- Wrong category code — c11 (re-parole) vs a12 (TPS) vs c19 (TPS pending) — these are NOT interchangeable
- Filing I-765 before re-parole (I-131) is approved — doesn't auto-grant work auth
- Wrong file format / file naming for online filing — USCIS rejects non-PDF or oversized files
- Believing "EAD will come automatically with re-parole" — separate adjudication required
- Missing employer I-9 reverification before EAD auto-extension expires

## I-821 (TPS)

- Confusing TPS dates — designation extends to October 19, 2026, but EAD auto-extension only goes to July 22, 2026
- Wrong category for re-registration vs initial filing
- Missing biometrics fee for first-time TPS filers
- Re-registering during the wrong window (windows are time-bound)
- Filing as TPS when not from a designated country

## I-912 (Fee Waiver)

- Insufficient evidence of inability to pay — USCIS requires specific documentation (means-tested benefit, household income, financial hardship)
- Missing income documentation — no recent paystubs / tax returns
- Filing fee waiver AFTER fee already paid (fees are not refunded retroactively)
- Not specifying the form being waived in Part 1
- Filing for a form that doesn't accept fee waivers (some forms like H-1B do not)

## G-1145 (E-Notification)

- Submitting AFTER USCIS already received the form — G-1145 must be IN the same package as the petition
- Stapling G-1145 to other forms (USCIS instructions specify it should be on top, attached)
- Missing email or phone (need at least one for e-notification to work)

## AR-11 (Address Change)

- Forgetting the deadline — must be filed within 10 days of moving (8 USC 1305)
- Filing for closed cases unnecessarily — only required if you have pending immigration matters
- Filing only AR-11 without also updating the address on each pending case (separate process)

## I-589 (Asylum)

- Missing the 1-year filing deadline (must file within 1 year of last arrival in US, with limited exceptions)
- Insufficient country conditions evidence — generic news articles often rejected
- Missing translator certification on foreign-language documents (8 CFR 103.2(b)(3) requires every foreign doc to have an English translation + translator competency certification)
- Discrepancies between asylum application and prior visa applications — DHS cross-references
- Filing without legal representation (asylum is high-stakes; pro se denial rates are significantly higher)
- Missing biographical detail — every address, every employment, every relative
